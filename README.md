## Getting Started

### Installation

Next.js framework
```bash
npx create-next-app@latest ./
```

<br>

Packages installed commands
```bash
npm install @headlessui/react cloudinary jsonwebtoken @types/jsonwebtoken graphql-request next-auth

npm install @grafbase/sdk --save-dev
```
